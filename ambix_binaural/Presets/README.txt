ambix_binaural_presets v0.1.4

This folder contains the presets including binaural loudspeaker impulse responses for ambix_binaural and ambix_decoder.

The content of this folder should go to:
(these folder are searched automatically, you can also define a custom search path in ambix_binaural or ambix_decoder)

OSX:
~/Library/ambix/binaural_presets/


Windows 7/8:
~/AppData/Roaming/ambix/binaural_presets/


Linux:
~/ambix/binaural_presets/
